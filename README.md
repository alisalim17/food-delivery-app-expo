## Getting Started

This is an example of how you may give instructions on setting up "bat" locally.
To get a local copy up and running follow these simple example steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/alisalim17/bat.git
   ```
2. Install packages
   ```sh
   yarn
   ```
3. Start the project
   ```sh
   yarn start
   ```
